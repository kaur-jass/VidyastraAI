# 🎓 VidyAstra AI

> An AI-powered learning platform that transforms educational content into intelligent, interactive, and personalized learning experiences using OCR, Computer Vision, Retrieval-Augmented Generation (RAG), and Large Language Models (LLMs).

---

## 📖 Overview

VidyAstra AI helps students learn more effectively from PDFs, notes, images, and study materials. The platform leverages modern AI technologies to extract knowledge, generate summaries, answer questions, create quizzes, and provide personalized learning assistance.

---

## ✨ Features

* 📄 OCR-Based Text Extraction
* 👁️ Vision-Based Content Understanding
* 📝 AI-Powered Summarization
* 🔍 Semantic Search
* 🤖 Intelligent Question Answering
* 🎯 Automatic Quiz Generation
* 📚 Personalized Learning Assistance
* ⚡ Batch Document Processing

## 🧠 AI Workflow

### 1. Content Upload

Users upload:

* PDFs
* Notes
* Images
* Lecture Slides
* Study Materials

### 2. OCR Processing

Extract textual information from scanned documents and images.

### 3. Vision Understanding

Analyze diagrams, charts, screenshots, and educational visuals using Vision Language Models.

### 4. Content Processing

Classify, organize, and structure educational content.

### 5. Embedding Generation

Convert content into vector embeddings for semantic retrieval.

### 6. Retrieval-Augmented Generation (RAG)

Retrieve relevant context from the knowledge base before generating responses.

### 7. AI Response Generation

Generate:

* Notes
* Summaries
* Explanations
* Quizzes
* Question Answers

## 🤖 Core AI Components

| Component       | Purpose                                   |
| --------------- | ----------------------------------------- |
| OCR Engine      | Text extraction from images and documents |
| Qwen2.5-VL      | Visual content understanding              |
| Embedding Model | Semantic representation of content        |
| Vector Database | Storage and retrieval of embeddings       |
| RAG Pipeline    | Context retrieval and grounding           |
| LLM             | Response generation and reasoning         |
